---\nname: misea-videocut-skills:智能分析\ndescription: 分析視頻並生成剪輯建議。轉錄、靜音檢測、精彩時刻、場景切換\nauthor: Misea1121\nsource: https://github.com/Misea1121/misea-videocut-skills\n---\n\n# 智能分析和剪輯建議 v1\n\n> 自動分析搞笑時刻、無理頭對話、驚喜反應，生成剪輯建議\n\n## 快速使用\n\n```\n用 misea-videocut-skills:智能分析，找出這個 Valorant 直播裡所有搞笑時刻和無理頭對話。\n用 misea-videocut-skills:智能分析，分析這個恐怖遊戲片段，找出所有驚嚇反應。\n```\n\n## 檢測類型\n\n### 1. 轉錄（Transcription）\n- **工具**：OpenAI Whisper（本地運行）\n- **支持語言**：中文、台語、英文混合\n- **輸出**：subtitles.srt 字幕文件\n\n### 2. 靜音檢測（Silence Detection）\n- ≤0.2s：忽略\n- 0.2-1s：標記\n- \\>1s：建議刪除\n\n### 3. 精彩時刻檢測（Highlight Detection）\n\n#### 搞笑時刻（Funny Moments）\n- 關鍵詞：「哈哈」、「天啊」、「操」、「笑死」、「怎麼可能」\n- 多人笑聲同時出現\n- 音量突然增加（尖叫、驚呼）\n\n#### 無理頭對話（Absurd Dialogue）\n- 突然的靜默（500ms+）+ 爆笑\n- 邏輯跳躍（「等等什麼」）\n- 隊友互相打斷\n\n#### 遊戲事件\n- **Valorant**：擊殺、死亡、勝負宣告\n- **Roblox**：遊戲 UI 變化、玩家反應\n- **恐怖遊戲**：尖叫聲、驚嚇 UI\n\n### 4. 場景切換檢測（Scene Change Detection）\n- 色彩變化\n- 鏡頭變化\n- 音頻轉場\n\n## 工作流\n\n```\n輸入：source.mp4 + audio.mp3\n    ↓\n1. 轉錄（Whisper）\n    → subtitles_words.json\n    ↓\n2. 靜音檢測\n    → silence_segments.json\n    ↓\n3. 精彩時刻檢測\n    → highlights.json\n    ↓\n4. 場景切換檢測\n    → scene_changes.json\n    ↓\n5. 風格分析（對比火紅視頻）\n    → style_suggestions.json\n    ↓\n6. 生成分鏡建議\n    ↓\n輸出：suggestions.html（分鏡頁面）\n```\n\n## 輸出格式\n\n```json\n{\n  \"transcription\": [...],\n  \"highlights\": [\n    {\n      \"start\": 12.5,\n      \"end\": 18.3,\n      \"type\": \"funny_moment\",\n      \"confidence\": 0.95,\n      \"keywords\": [\"哈哈\", \"天啊\"],\n      \"reason\": \"多人笑聲 + 關鍵詞匹配\"\n    }\n  ],\n  \"scene_changes\": [...],\n  \"suggestions\": [...]\n}\n```\n\n## 腳本\n\n- `transcribe_tw.js` - 中文/台語轉錄\n- `silence_detector.js` - 靜音檢測\n- `highlight_detector.js` - 精彩時刻檢測\n- `scene_change.js` - 場景切換檢測\n- `style_analyzer.js` - 風格分析\n- `auto_cut_suggestion.js` - 生成分鏡建議\n"
